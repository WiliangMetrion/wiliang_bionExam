## Getting Started

Please make sure you have the following requirement installed :

- Flutter SDK 3.7 >=
- Dart SDK
- Android Studio
- Android Emulator API 35

You can download or clone this project from github.

Please run the following command before run it :

- flutter clean
- flutter pub get
- flutter run
